## 🚀 Free Deployment Options

### Option 1: Vercel (Recommended — Fastest)
1. Go to [vercel.com](https://vercel.com) and sign in with GitHub
2. Click **"New Project"** → **"Import Git Repository"**
3. Upload this folder OR drag & drop on Vercel dashboard
4. Click **Deploy** — Done! ✅
5. Your link: `https://your-project-name.vercel.app`

### Option 2: Netlify (Easiest Drag & Drop)
1. Go to [netlify.com](https://netlify.com) and sign in
2. Go to **Sites** → drag & drop this entire folder into the page
3. Your site is live in seconds! ✅
4. Your link: `https://random-name.netlify.app`
5. Rename: Site settings → Change site name → `aditya-priya-wedding.netlify.app`

### Option 3: GitHub Pages (Free Forever)
1. Create a new GitHub repository named `wedding-invite`
2. Upload all files to the repository
3. Go to **Settings** → **Pages** → **Source: main branch / root**
4. Your link: `https://yourusername.github.io/wedding-invite`

## ✏️ Customization Guide

Open `index.html` and find these sections to update:

| What to Change | Search For |
|---|---|
| Couple Names | `आदित्य` / `प्रिया` |
| Wedding Date | `२३ फेब्रुवारी २०२५` |
| Venue | `श्री रामकृष्ण मंगल कार्यालय` |
| Father's Name (Groom) | `राजेंद्र देशपांडे` |
| Father's Name (Bride) | `दिनेश सावंत` |
| Phone Numbers | `+91 98765 43210` |
| Maps Link | `maps.google.com/?q=...` |
| Countdown Date | `new Date('2025-02-23T11:00:00')` |
